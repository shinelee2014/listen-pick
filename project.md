## 🎯 1. 项目背景与目标

*   **项目愿景**：为江苏省苏州市小学生（1-6年级）设计的“听句子选图片/Emoji”英语听力练习应用。要求界面精美（马卡龙色调，防 iPad 横屏溢出），选项纯 Emoji 显示（无单词干扰以纯考听力），支持免密多孩子登录和进度隔离，方便部署在本地威联通 NAS 和 GitHub Pages 上。
*   **当前核心任务**：
    *   配合用户开启 GitHub Pages 静态站点。
    *   辅助在威联通 (QNAP) 局域网上完成部署和测试。

## 🛠️ 2. 技术栈与核心依赖

*   **运行环境**：纯前端静态网页 (SPA)，可运行于任何现代浏览器，支持离线运行。
*   **核心库/框架**：
    *   **Vanilla JS**: 用于核心答题引擎、路由与状态管理。
    *   **canvas-confetti**: 答对与通关时的粒子彩带特效（已本地化保存至 `js/libs/`，支持完全离线）。
*   **第三方外部服务/API**：
    *   **Web Speech API (speechSynthesis)**：浏览器原生 TTS 发音引擎，零费用、零 Token 消耗、支持局域网离线高音质发音。

## 🏗️ 3. 架构设计与模块分工

*   **目录分层**：
    *   `css/style.css`：马卡龙护眼渐变配色，iPad (1024x768) 屏幕防溢出适配，微动效。
    *   `data/`：包含 `grade1.json` 至 `grade6.json` 共 600 道苏州小学英语课程同步句型。
    *   `images/`：备用卡通 Chibi 图片资产（如 girl, boy, cat, dog 等）。
    *   `js/storage.js`：本地 localStorage 存储模块，管理设置并按孩子账户名隔离进度。
    *   `js/audio.js`：封装 Web Speech 语音播放逻辑，支持慢速/正常/快速调节。
    *   `js/app.js`：主控制器，处理答题判定、星星飞入动画、彩带粒子控制、视图路由及账户增删。
    *   `index.html`：单页面主控结构，包含模态弹窗和页面骨架。
