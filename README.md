# Listen & Pick (听句子，选图片) 🎧

一个专为江苏省苏州市小学生（1-6年级）设计的趣味英语听力练习网页应用。

支持在浏览器中直接运行，也非常适合部署在 GitHub Pages 以及家庭局域网威联通 (QNAP) NAS 上，让孩子们在平板或手机上随时开始学习。

---

## ✨ 应用特色

*   **📚 同步苏州小学英语教材**：
    *   内置 1-6 年级全量关卡（每个年级 100 关，共计 600 道语境关卡）。
    *   关卡词汇与句型深度适配苏教版/译林版英语教材大纲。
*   **🎨 专为儿童打造的马卡龙美学**：
    *   采用柔和清新的护眼渐变配色，界面生动活泼。
    *   **平板优化**：适配 iPad 横屏（1024x768）触控尺寸，首屏显示无滚动条，防溢出设计。
*   **🦄 纯 Emoji 听力测验**：
    *   首创 **Emoji-First** 纯图案渲染逻辑，选项卡片下方无任何单词干扰。
    *   强迫孩子使用纯听力与场景联想进行选择，极大提升听力理解效果。
*   **👥 免密多孩子账户系统**：
    *   支持为家里的多个宝贝分别建档（如小明、小华），每个账号拥有独立的通关星星数和进度存取（数据安全隔离）。
    *   登录时只需选择自己的萌宠头像，无需记忆繁琐的密码。
    *   在关卡地图页提供一键“重置当前孩子进度”功能，方便开启下一轮复习。
*   **⭐ 动感反馈与彩带庆祝**：
    *   答对时，星星会以平滑的弧线飞入右上角计分栏，并带有计分栏弹跳和局部的 `canvas-confetti` 七彩纸屑爆破。
    *   每天的 5 关全部完成时，会触发华丽的双侧彩带喷泉连续喷洒，并授予 1~3 颗星的评级。
    *   答错时，卡片会产生可爱的左右颤抖反馈，并高亮正确选项、自动重播语音。
*   **🔌 100% 离线 & 极简部署**：
    *   使用浏览器原生 **Web Speech API** 发音，支持慢速/正常/快速调节，零费用、零 Token 消耗、完全不需要外网。
    *   纯静态网页 (SPA)，不需要 Node.js、Docker 或任何数据库，复制文件夹即部署成功。

## 🚀 部署与使用方法

### 1. 本地直接使用
双击双击 `index.html` 即可在浏览器中打开使用。
如需使用本地 HTTP 服务运行，可在项目根目录下运行：
```bash
python -m http.server 8080
```
然后在浏览器访问 `http://localhost:8080`。

### 2. 部署在 GitHub Pages
1. 将本项目推送至您的 GitHub 仓库（例如 `shinelee2014/listen-pick`）。
2. 在 GitHub 仓库页面点击 **Settings** -> **Pages**。
3. 将 **Branch** 选择为 `main` 分支下的 `/(root)`，点击 **Save**。
4. 部署成功后即可通过 `https://[您的用户名].github.io/listen-pick/` 在线访问。

### 3. 部署在威联通 (QNAP) 局域网 NAS
1. 在 QNAP 控制台的 **“应用服务”** -> **“网页服务器” (Web Server)** 中，启用 Web 服务器。
2. 打开 File Station，将整个项目文件夹（或 GitHub 下载的 ZIP 压缩包）解压并拷贝到默认共享文件夹 **`Web`** 下的 `listen-pick` 目录中。
3. 局域网内的平板、手机通过浏览器直接访问 `http://[QNAP的局域网IP]/listen-pick/` 即可完美离线运行！
